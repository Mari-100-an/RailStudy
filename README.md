철도 기관사 지망생을 위한 제2종 전기차량운전면허 시험 대비 웹 학습 플랫폼입니다.

## 📌 프로젝트 개요

- **목표**: 제2종 전기차량운전면허 시험 합격을 위한 효율적인 학습 도구 제공
- **대상 사용자**: 도시철도 기관사 지망생
- **특징**: Focus Mode(집중 학습)와 Game Mode(게이미피케이션) 두 가지 UI 모드 지원

## ✅ 현재 구현된 기능

### 1. 문제 풀이 시스템
- **과목별 학습**: 철도관련법, 도시철도시스템, 전기동차 구조, 운전이론, 비상시 조치
- **랜덤 문제 출제**: 선택한 개수만큼 무작위 문제 출제
- **자동 채점**: 실시간 정답 확인 및 해설 표시
- **출처 인용**: 각 문제에 표준교재/법령 출처 명시

### 2. 오답 노트
- **자동 저장**: 틀린 문제 자동 기록
- **과목별 필터링**: 특정 과목의 오답만 확인 가능
- **오답 재학습**: 오답 노트에서 바로 다시 풀기
- **마스터 처리**: 정답 시 오답 목록에서 자동 제거

### 3. 학습 대시보드
- **전체 통계**: 총 학습 문제 수, 평균 정답률, 학습 일수, 최장 연속 정답
- **과목별 성취도 차트**: Chart.js 기반 막대그래프
- **주간 학습량 차트**: 최근 7일간 학습 추이 라인그래프

### 4. UI 모드 전환
- **Focus Mode (집중 모드)**
  - 미니멀리즘 디자인
  - 큰 글씨, 넓은 여백
  - 애니메이션 최소화
  - 가독성 최우선
  
- **Game Mode (게임 모드)**
  - 다크 네이비 테마
  - XP 및 레벨 시스템
  - 콤보 보너스
  - 뱃지 수집
  - 파티클 효과

### 5. 게이미피케이션 요소
- **레벨 시스템**: XP 획득으로 레벨업
- **콤보 보너스**: 연속 정답 시 추가 XP
- **16종 뱃지**: 다양한 조건 달성 시 획득
  - 첫 발걸음, 열정의 시작, 지식 탐험가, 학습 마스터
  - 연속 킬러, 콤보 마스터, 무적 기관사
  - 과목별 전문가 뱃지 등

### 6. 데이터 관리
- **LocalStorage 저장**: 학습 진도 자동 저장
- **데이터 내보내기**: JSON 형식 백업
- **데이터 초기화**: 전체 학습 기록 리셋
