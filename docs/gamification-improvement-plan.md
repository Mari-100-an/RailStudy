# 게이미피케이션 개선 계획

## 1. 레벨 시스템 개선 (총 2000-3000문제 기준)

### 현재 문제점
- Level 20이 최고 → 너무 낮음
- 레벨업 곡선이 단순 (1.5 지수)

### 개선안
- **최대 레벨: 100** (그랜드 마스터)
- **단계별 등급 시스템**:
  - Lv 1-10: 🌱 견습생 (Trainee)
  - Lv 11-20: 🔧 초급 기관사 (Junior)
  - Lv 21-35: ⚙️ 중급 기관사 (Intermediate)
  - Lv 36-50: 🚃 상급 기관사 (Senior)
  - Lv 51-70: 🚄 전문 기관사 (Expert)
  - Lv 71-85: 🌟 마스터 기관사 (Master)
  - Lv 86-99: 👑 그랜드 마스터 (Grand Master)
  - Lv 100: 💎 레전드 기관사 (Legend)

- **XP 곡선 조정**:
  - Lv 1-20: 빠른 성장 (초반 동기부여)
  - Lv 21-50: 중간 속도
  - Lv 51-100: 느린 성장 (장기 목표)

## 2. 배지 시스템 확장 (50개 이상)

### 카테고리별 배지

#### A. 진행도 배지 (Progress Badges)
- 첫 걸음 (1문제)
- 초보 학습자 (10문제)
- 열정적인 학습자 (50문제)
- 백 문제 정복 (100문제)
- 삼백 문제 마스터 (300문제)
- 오백 문제 챔피언 (500문제)
- 천 문제 전설 (1000문제)
- 이천 문제 신화 (2000문제)
- 완벽주의자 (전체 문제 100% 정답)

#### B. 과목별 전문가 배지
각 과목당 3단계:
- 입문 (20문제 정답)
- 숙련 (50문제 정답)
- 마스터 (100문제 정답)
- 그랜드 마스터 (200문제 정답)

#### C. 정확도 배지
- 정밀 사수 (90% 이상, 50문제)
- 명사수 (95% 이상, 100문제)
- 완벽주의 (98% 이상, 200문제)
- 백발백중 (100%, 10문제)

#### D. 연속 배지 (Streak Badges)
- 3연속, 5연속, 10연속, 15연속, 20연속, 30연속, 50연속, 100연속

#### E. 시간 기반 배지
- 아침형 인간 (오전 6-9시 학습)
- 야행성 학습자 (밤 10시-새벽 2시 학습)
- 주말 전사 (주말 학습)
- 월요병 극복 (월요일 학습)
- 일주일 챌린저 (7일 연속)
- 한 달 마라토너 (30일 연속)

#### F. 특별 배지
- 빠른 학습자 (1시간에 50문제)
- 철도 박사 (모든 과목 마스터)
- 오답 정복자 (오답 100개 복습)
- 완벽한 세션 (한 세션 100% 정답)

## 3. 재미 요소 추가 (깔끔하게)

### 3.1 일일 목표 시스템
- 매일 목표 자동 생성 (10문제, 20문제 등)
- 달성 시 보너스 XP
- 연속 달성 시 추가 보너스

### 3.2 주간 챌린지
- 주간 목표: 100문제 풀기
- 특정 과목 집중 주간
- 달성 시 특별 배지

### 3.3 마일스톤 보상
- 100문제마다 특별 보상
- 레벨업 시 축하 애니메이션
- 등급 승급 시 특별 효과

### 3.4 통계 시각화 개선
- 과목별 레이더 차트
- 일별 학습 히트맵
- 진행도 프로그레스 바
- 레벨 프로그레스 로드맵

### 3.5 칭호 시스템
- 레벨/배지에 따라 자동 획득
- 칭호 선택 기능
- 홈 화면에 표시

## 4. 구현 우선순위

1. **Phase 1**: 레벨 시스템 확장 (Lv 100)
2. **Phase 2**: 배지 시스템 확장 (50개)
3. **Phase 3**: 일일 목표 시스템
4. **Phase 4**: 칭호 시스템
5. **Phase 5**: 주간 챌린지
