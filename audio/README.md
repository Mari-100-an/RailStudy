# 오디오 파일 가이드

## 효과음 파일 (audio/sfx/)

다음 효과음을 다운로드하여 `audio/sfx/` 폴더에 넣어주세요:

### 필수 파일:
- `correct.mp3` - 정답 소리 (밝은 종소리, 벨 소리)
- `wrong.mp3` - 오답 소리 (부드러운 에러음, 낮은 톤)
- `levelup.mp3` - 레벨업 (축하 팡파르, 상승 멜로디)
- `combo.mp3` - 콤보 (짧은 틱, 클릭 소리)
- `select.mp3` - 선택 (가벼운 클릭)

### 선택 파일 (배지 티어별):
- `badge-bronze.mp3` - 브론즈 배지
- `badge-silver.mp3` - 실버 배지
- `badge-gold.mp3` - 골드 배지
- `badge-platinum.mp3` - 플래티넘 배지
- `badge-diamond.mp3` - 다이아몬드 배지

**또는** `badge.mp3` 하나만 사용해도 됩니다.

## 배경음악 (audio/bgm/)

- `lofi-study.mp3` - 로파이 비트 (2-3분, 루프 가능한 곡)

## 추천 다운로드 사이트

### 효과음:
1. **Pixabay** - https://pixabay.com/sound-effects/
   - 검색어: "correct", "success", "level up", "click", "combo"
   
2. **Freesound** - https://freesound.org/
   - 무료 회원가입 필요
   - 검색어: "achievement", "bell", "error", "tick"

3. **Mixkit** - https://mixkit.co/free-sound-effects/
   - 검색어: "notification", "game", "ui"

### 배경음악:
1. **Pixabay Music** - https://pixabay.com/music/
   - 검색어: "lofi", "study", "chill", "ambient"

2. **YouTube Audio Library**
   - 무료, 고품질

## 파일 요구사항

- 형식: MP3 (권장) 또는 WAV
- 효과음 크기: 50KB ~ 500KB
- BGM 크기: 3MB ~ 5MB (2-3분)
- 샘플레이트: 44.1kHz 또는 48kHz

## 라이선스 확인

- **상업적 이용 가능** 파일인지 확인
- 일부 사이트는 크레딧 표기 필요 (앱 정보에 추가)
